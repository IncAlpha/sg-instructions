Как собрать проект и опубликовать

Пошагово: установка, сборка, публикация. 10 минут.

15 августа 2025 · Василий Петров

Что понадобится

  • Установленный Node.js 18+
  • Аккаунт GitHub
  • Статический хостинг (GitHub Pages / Netlify)

Шаги

  1. Клонируйте репозиторий.
  2. Установите зависимости и соберите: 
    npm ci
npm run build
  3. Загрузите сборку на хостинг.
Совет: для коротких клипов используйте GIF или MP4 без звука — Telegram IV поддерживает анимированные изображения.

Если что-то пошло не так

Проверьте версии Node.js и права на репозиторий.
